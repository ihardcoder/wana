# Waterloo-analytics

### 前端数据统计，包括以下方面的数据统计：
1. 性能数据：通过performance API获取；
2. js异常数据（todo）；
3. 设备信息（todo）

### 部署流程

* 下载源码：
```
git clone git@github.com:ihardcoder/Waterloo-analytics.git
```
* 安装依赖
```
npm install
```
* 编译
```
npm run build
```
> 也可以使用对应的cli工具[Waterloo-analytics-cli](https://github.com/ihardcoder/Waterloo-analytics-cli)。
